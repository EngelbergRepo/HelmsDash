# HelmsDash

A medieval endless runner built with Three.js. Sprint through cobblestone streets as an armored knight, dodge carts and obstacles, collect gold coins, and grab power-ups — how far can you go?

Built as an entry for **[VibeJam 2026](https://vibej.am/2026)**. More than 90% of the game code and assets were generated by AI.

---

## Install & Run

**Requirements:** Node.js 18+

```bash
cd HelmsDash
npm install
npm run dev
```

Open **http://localhost:5173** in your browser.

## Build for Production

```bash
npm run build    # output → HelmsDash/dist/
npm run preview  # serve the production build locally
```

## Tech Stack

- [Three.js](https://threejs.org/) r165
- [Vite](https://vitejs.dev/) 5
- [Howler.js](https://howlerjs.com/) — audio
- Vanilla ES modules, no framework
